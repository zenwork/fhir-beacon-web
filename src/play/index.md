# Play

Play is the planned full-view editor for trying FHIR Beacon examples.

The goal is to let users load examples, edit code, data, or configuration, and
see the rendered view.

## Preview Status

The editor is not part of the first public preview.

For now, use:

- [Storybook](https://fhir-beacon.deno.dev) for component-level examples
- [Showcase app](https://fhir-beacon-app.deno.dev) for broader exploration
