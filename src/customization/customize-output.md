# Customizing Output

This section provides guidance on how to customize the output of the FHIR Beacon service to meet your presentation needs


## Editing Templates

All components come with a model compliant output. When this is not good enough then you have a few choices:

1. replace the presentation components with your own 
2. Extend in HTML the fhir-beacon elements and implment a custom rendering of an element
3. Extend in JS the fhir-beacon classes and implment a custom rendering of an element
4. Extend the data element take full control of the UX while retaing the full life-cycle
